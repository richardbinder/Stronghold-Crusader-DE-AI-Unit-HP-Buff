# Stronghold Crusader DE AI Buff

This is a mod for Stronghold Crusader Definitive Edition that buffs AI resource income, unit HP, and unit damage.

The resource, HP, and damage multipliers can be configured in the lobby mod options. Works both in Singleplayer Skirmish and Multiplayer as long as all players have the mod installed.

Requires [Stronghold Crusader DE Script Extender](https://www.nexusmods.com/strongholdcrusaderdefinitiveedition/mods/35) and [BepInEx 5 Bootstrapper](https://www.nexusmods.com/strongholdcrusaderdefinitiveedition/mods/36) to work.
